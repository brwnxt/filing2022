Laravel Basic

Laravel Basic merupakan salah satu framework yang mampu memaksimalkan penggunaan bahasa pemrograman PHP dalam proses pengembangan website, sehingga mempermudah developer website dalam menyelesaikan pekerjaan.

Pada workshop Laravel Basic ini ini akan mempelajari Pengenalan Framework Laravel , Object Oriented Programming , Desain Entity Relationship Diagram, Pembuatan website dinamis , Blade Template Laravel, Database & Migration Laravel, CRUD, Laravel libraries dan Bahasa PHP & SQL

Dari pokok-pokok materi ini di harapkan peserta workshop dapat mengetahui dan memahami pengenalan Framework Laravel dan konsep Model View Controller serta peserta dapat mengembangkan website dengan menggunakan Framework Laravel.
